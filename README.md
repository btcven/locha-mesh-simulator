# lochaMesh radio link simulator




## Getting started

### Requirements:

    - Node.js v10.15.x

### Installing

Clone the repo.
```bash
$ git clone https://gitlab.com/btcven/xxx
```

Go into the cloned folder
```bash
cd locha-simulator

```

Install dependencies:
```bash
npm install

```

Start the simulation:
```bash
npm start

```


Open your browser and go to http://localhost:3000
